# تحليل بيانات ركاب تايتانك 🚢

مشروع تحليل بيانات أول لي باستخدام Python و Pandas.

## وصف المشروع
في المشروع دا حللت بيانات ركاب سفينة تايتانك عشان أعرف شنو العوامل اللي أثرت على فرص النجاة.
الهدف منه تعلم تحليل البيانات من الصفر خطوة بخطوة.

## الملفات الموجودة
| الملف | الوصف |
| --- | --- |
| `titanic.ods` | ملف البيانات الأصلي فيه معلومات 891 راكب |

## الأدوات المستخدمة
- Python 3.11
- Pandas 2.0 
- Jupyter Notebook
- Excel/ODS

## أهم النتائج الأولية
| الفئة | ملاحظة |
| --- | --- |
| النساء | نسبة النجاة أعلى بكثير |
| الأطفال | ليهم أولوية في النجاة |
| الدرجة الأولى | فرص النجاة أعلى |

## كيف تشغل المشروع
1. نزل ملف `titanic.ods`
2. افتحه بـ Excel أو Pandas
3. ابدأ التحليل
