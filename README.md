# تحليل بيانات ركاب تايتانك Titanic

مشروع تحليل بيانات أول لي باستخدام Python و Pandas.

## وصف المشروع
في المشروع دا حللت بيانات ركاب سفينة تايتانك عشان أعرف شنو العوامل اللي أثرت على فرص النجاة.
الهدف منه تعلم تحليل البيانات خطوة بخطوة.

## الملفات الموجودة
- `titanic.ods` : ملف البيانات الأصلي فيه معلومات الركاب

## الأدوات المستخدمة
- Python
- Pandas 
- Jupyter Notebook
